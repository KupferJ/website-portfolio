Movie Application (React)

Description

This is the frontend of a movie list application which pulls its data from a personally created movie API.
The movie app was designed to give users access to information about a list of movies as well as some key data about them, namely their lead director, genre and a plot description.
Users are able to sign up, log in/out, update their personal information and add movies as favorites.

The application was built using HTML, CSS, JavaScript, React, React Bootstrap, React Redux and Parcel.

Meet Application

Description

This Meet app was built as a serverless, progressive web application (PWA). Using the Google Calendar API, the application fetches data about a number of upcoming, coding-related events in a number of different countries. Users can log in using their google account to access the data, choose the number as well as the location of the events they want displayed and access a short description of them.

The application was built with React using a test-driven development (TDD) approach.

Chat Application

Description

This mobile Chat application was built as a progressive web application (PWA). Using Firestore as its database, users can select their name and a background color to then enter the chatroom to chat with other people online at the time.
Users are given the ability to allow the application access to their mobile phones' functions in order to send their current location in chat. They can also permit access to their phone's library and camera to take and send a picture.

The application was built using React Native, Expo, Firebase and the Gifted Chat library.
