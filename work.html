Description

This movie app was designed to give users access to information about a list of movies as well as some key data about them, namely their lead director, genre and a plot description.
Users are able to sign up, log in/out, update their personal information and add movies as favorites.

Meet Application

Description

This Meet app was built as a serverless, progressive web application (PWA). Using the Google Calendar API, the application fetches data about a number of upcoming, coding-related events in a number of different countries. Users can log in using their google account to access the data, choose the number as well as the location of the events they want displayed and access a short description of them.
The application was built with React using a test-driven development (TDD) approach.
